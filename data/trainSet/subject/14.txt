				<p>?????? What a Race! 这是看完在珠海举行的洲际勒芒大赛的老外们最多说的一句话！第一次在中国（珠海）举行的勒芒洲际杯大赛确实是非常精彩，标致和奥迪两支厂商车队都使出了浑身解数，让我打开了眼界。一个长达1000公里，200多个圈的比赛，到最后冠军的标致只险胜亚军的奥迪不到5秒！在此可以看到这个比赛是多么的高水平。事实上不光是我，一众长年在珠海赛车场，看惯大场面的赛车人都被整震撼了。为了这次比赛，就标致一个车队来说，整个标致运动部有60多人从法国飞来了珠海。除了两台908赛车和两台展示用的RCZ赛车是空运到珠海外，还有5个集装箱的物资（不含轮胎和燃油）从海路运到。我相信另一边的奥迪车队的排场也不遑多让的。事实上这次珠海勒芒比赛也让大家看到厂商车队的技术实力，像在第一天的试车中，标致908便已创出了珠海赛车场有史以来的最快单圈时间 C 1分20秒！这个成绩绝对是超出了所有珠海赛车人的想象，也让大家无法不对勒芒赛车另眼相看。原来珠海赛车场的旧单圈记录是由A1赛车在排位赛中创出的，但勒芒赛车在珠海的第一次试车便比A1快了3秒多！如果不是排位赛受到了天雨的影响，我绝对相信勒芒LMP1可以跑进1分19甚至18秒以下。1分20秒的圈时是什么概念呢？举个例子- 在同一天举行的保时捷卡雷拉杯的单圈成绩是1分39秒，两者相差达19秒！而今年CTCC在珠海站2000组的成绩 C 头位福特福克斯是1分50秒！?</p>
<p style="text-align: center"><img class="size-large wp-image-70 aligncenter" src="http://blog.che168.com/suhualong/files/2010/11/lemans-600x478.jpg" alt="" width="600" height="478" /></p>
<p>??????? 勒芒比赛虽然精彩，但在中国懂得欣赏的人好像还不是太多。在比赛当天主看台的上座率好像50%也不到，这样的情况无不让人担心珠海勒芒比赛会步Moto GP、澳大利V8超级赛车和A1世界赛的后尘，举办一、两届后便支持不下去。我在比赛期间曾经和主办比赛的官员聊过，他们表示上座率低是由于这次没有送票，因此他们已经预知现场的观众人数不会太理想。而决定不送票的原因是希望养成中国观众的购票习惯，因为一旦送票以后便很难让观众再购票看比赛了。。。我指出票价达150元的门票绝对不是一般中国民众可以负担的价钱，我也很希望看到中国车迷购票进场，因为愿意付钱看比赛的才是真车迷，可惜太高昂的门票价格直接打击了车迷进场看比赛的愿望。我希望勒芒大赛组织者能吸收教训，在下次来华比赛的时候可以大幅降低票价，然更多的车迷可以欣赏到这个独一无二，无以伦比的比赛。</p>

    function reply(authorId, commentId, commentBox) {
    	var author = document.getElementById(authorId).innerHTML;
    	var insertStr = '<a href="#' + commentId + '">@' + author.replace(/\t|\n|\r\n/g, "") + ' </a> \n';
    	$("#parent_id").val(commentId);

        	
    	appendReply(insertStr, commentBox);
    }
    
    function quote(authorId, commentId, commentBodyId, commentBox) {
    	var author = document.getElementById(authorId).innerHTML;
    	var comment = document.getElementById(commentBodyId).innerHTML;
    
    	var insertStr = '<blockquote cite="#' + commentBodyId + '">';
    	insertStr += '\n<strong><a href="#' + commentId + '">' + author.replace(/\t|\n|\r\n/g, "") + '</a> :</strong>';
    	insertStr += comment.replace(/\t/g, "");
    	insertStr += '</blockquote>\n';
    
    	insertQuote(insertStr, commentBox);
    }
    
    function appendReply(insertStr, commentBox) {
    	if(document.getElementById(commentBox) && document.getElementById(commentBox).type == 'textarea') {
    		field = document.getElementById(commentBox);
    
    	} else {
    		alert("The comment box does not exist!");
    		return false;
    	}
    
    	if (field.value.indexOf(insertStr) > -1) {
    		alert("You've already appended this reply!");
    		return false;
    	}
    
    	if (field.value.replace(/\s|\t|\n/g, "") == '') {
    		field.value = insertStr;
    	} else {
    		field.value = field.value.replace(/[\n]*$/g, "") + '\n\n' + insertStr;
    	}
    	field.focus();
    }
    
    function insertQuote(insertStr, commentBox) {
    	if(document.getElementById(commentBox) && document.getElementById(commentBox).type == 'textarea') {
    		field = document.getElementById(commentBox);
    
    	} else {
    		alert("The comment box does not exist!");
    		return false;
    	}
    
    	if(document.selection) {
    		field.focus();
    		sel = document.selection.createRange();
    		sel.text = insertStr;
    		field.focus();
    
    	} else if (field.selectionStart || field.selectionStart == '0') {
    		var startPos = field.selectionStart;
    		var endPos = field.selectionEnd;
    		var cursorPos = startPos;
    		field.value = field.value.substring(0, startPos)
    					+ insertStr
    					+ field.value.substring(endPos, field.value.length);
    		cursorPos += insertStr.length;
    		field.focus();
    		field.selectionStart = cursorPos;
    		field.selectionEnd = cursorPos;
    
    	} else {
    		field.value += insertStr;
    		field.focus();
    	}
    }
